# কুইজ মাস্টার — পাবলিশ ও ফেসবুকে শেয়ার গাইড

---

## ধাপ ১: GitHub-এ আপলোড করুন

1. **github.com** এ যান → Sign up / Login করুন
2. ডান পাশে **"+"** বাটনে ক্লিক করুন → **"New repository"**
3. Repository নাম দিন: `quiz-master`
4. **"Create repository"** চাপুন
5. **"uploading an existing file"** লিংকে ক্লিক করুন
6. ZIP ফাইল আনজিপ করুন, তারপর `quiz-master` ফোল্ডারের **ভেতরের সব ফাইল** ড্র্যাগ করে আপলোড করুন:
   ```
   ফোল্ডার স্ট্রাকচার:
   ├── package.json
   ├── public/
   │   ├── index.html
   │   └── og-image.png
   └── src/
       ├── index.js
       └── App.js
   ```
7. **"Commit changes"** চাপুন

---

## ধাপ ২: Vercel-এ ডিপ্লয় করুন (ফ্রি!)

1. **vercel.com** এ যান → **"Sign Up"** → **GitHub দিয়ে লগইন**
2. **"Add New Project"** চাপুন
3. আপনার `quiz-master` repository সিলেক্ট করুন
4. Framework Preset: **Create React App** সিলেক্ট করুন
5. **"Deploy"** চাপুন
6. ২-৩ মিনিট অপেক্ষা করুন ✅
7. আপনার সাইট লাইভ হবে! যেমন: **https://quiz-master-xyz.vercel.app**

---

## ধাপ ৩: অ্যাডমিন প্যানেলে প্রশ্ন যোগ করুন

1. আপনার সাইটে যান
2. নিচে **"🔐 অ্যাডমিন প্যানেল"** বাটনে ক্লিক করুন
3. পাসওয়ার্ড দিন: `admin123`
4. **"➕ প্রশ্ন"** ট্যাবে নতুন প্রশ্ন যোগ করুন:
   - বিষয় সিলেক্ট করুন
   - বোর্ডের নাম দিন
   - প্রশ্ন বাংলায় লিখুন
   - ৪টি অপশন দিন
   - সঠিক উত্তরে ক্লিক করুন (সবুজ হবে)
   - 🏆 পুরস্কার শিরোনাম দিন (যেমন: "ব্যাংকিং বিশেষজ্ঞ 🏦")
   - ব্যাখ্যা লিখুন
5. **"✅ সংরক্ষণ"** চাপুন

## ধাপ ৪: ফেসবুকে প্রশ্নের লিংক শেয়ার করুন

### লিংক কপি করুন:
1. অ্যাডমিন প্যানেলে যান
2. **"📋 ম্যানেজ"** ট্যাবে যান
3. যে প্রশ্নটি শেয়ার করতে চান তার পাশে **"🔗 লিংক কপি"** চাপুন
4. লিংক কপি হয়ে যাবে! যেমন:
   ```
   https://quiz-master-xyz.vercel.app?q=q1234567890
   ```

### ফেসবুক পোস্ট/অ্যাড তৈরি করুন:

**ফ্রি পোস্ট (Facebook Page/Group):**
```
🧠 কুইজ চ্যালেঞ্জ!

'NSF' এর পূর্ণরূপ কী? 🤔
সঠিক উত্তর দিলে পাবেন "ব্যাংকিং বিশেষজ্ঞ 🏦" উপাধি!

⏱️ ২৫ সেকেন্ডে উত্তর দিতে হবে!
👇 এখনই খেলুন:
https://quiz-master-xyz.vercel.app?q=q1

#HSC2025 #হিসাববিজ্ঞান #বোর্ডপরীক্ষা #MCQ
```

**Facebook Ad Caption (Bangla):**
```

এই MCQ-তে সঠিক উত্তর দিয়ে বিশেষ উপাধি জিতুন! 🏆
✅ বোর্ড পরীক্ষার প্রশ্ন
✅ সময়ভিত্তিক চ্যালেঞ্জ
✅ সঠিক উত্তরে ব্যাখ্যা + টাইটেল

👇 এখনই চ্যালেঞ্জ নিন:
[প্রশ্নের লিংক এখানে পেস্ট করুন]
```

### Facebook Ad Setup:
1. **Facebook Ads Manager** (business.facebook.com) এ যান
2. **"Create"** → Objective: **"Traffic"** সিলেক্ট করুন
3. Audience: বাংলাদেশ, বয়স ১৬-২০, Interest: "HSC", "Board Exam"
4. Ad Format: **Single Image** বা **Carousel**
5. Website URL: আপনার প্রশ্নের লিংক পেস্ট করুন
6. Primary Text: উপরের ক্যাপশন ব্যবহার করুন

## ধাপ ৫: অ্যানালিটিক্স দেখুন

1. অ্যাডমিন প্যানেলে লগইন করুন
2. **"📊 রিপোর্ট"** ট্যাবে যান
3. দেখতে পাবেন:
   - 👥 মোট কতজন শিক্ষার্থী উত্তর দিয়েছে
   - ✓ কতজন সঠিক উত্তর দিয়েছে
   - ✗ কতজন ভুল উত্তর দিয়েছে
   - 📊 প্রশ্নভিত্তিক সঠিকতার হার (%)

## গুরুত্বপূর্ণ নোট

- **পাসওয়ার্ড পরিবর্তন:** App.js ফাইলে `ADMIN_PASS = "admin123"` খুঁজে আপনার পাসওয়ার্ড দিন
- **প্রতিটি প্রশ্নের আলাদা লিংক** — শিক্ষার্থীরা শুধু সেই একটি প্রশ্নই দেখবে
- **বিষয় অনুযায়ী থিম বদলায়** — হিসাববিজ্ঞান নীল, ফিন্যান্স বেগুনি, ব্যবস্থাপনা সবুজ
- **মোবাইল ও পিসি** দুটোতেই কাজ করে
